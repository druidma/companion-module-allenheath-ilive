# Allen & Heath iLive TCP module

Controls the Allen & Heath iLive.

## Functions:


* Mute Input Channel
* Mute FX Sends and Returns
* Mute DCA
* Mute Mix Buses
* Set Preamp 48v
* Set fader levels for Mix Buses, FX Sends and Returns, DCAs and Inputs
* Recall Scene

> Current version: 1.0.0
